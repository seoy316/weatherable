# weatherable

## **프로젝트 소개**
**: ** <br>

**주요 기능**
- 방문 장소 태깅 업로드
- 기상 정보 및 현재 위치에 따른 장소 추천
  
<br>

## 구현 기능

**1. 회원 관련 기능**
- 회원가입
- 로그인/로그아웃
- 회원 선호도 폼 입력

**2. 추천시스템**
   - 추천 알고리즘

**3. SNS**
   - 게시글 업로드
   - 게시글 삭제
<br>
